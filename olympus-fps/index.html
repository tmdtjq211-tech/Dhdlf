[ TACTICAL MAZE (전술 미로 교전) ]

이번 작전 구역은 막힌 벽들로 둘러싸인 복잡한 미로(복도) 형태입니다.
★ 은폐/엄폐: 훈련용 연두색 벽에 몸을 숨기면 적의 시야에서 벗어납니다. 복도를 순찰하는 적과 마주치면 즉각 대응사격 하세요!


[ PERFECT HIT 시스템 ]

적의 머리(Head)나 가슴 정중앙(Chest)을 정확히 쏘면 PERFECT!가 뜨며 일반 적은 즉사, 보스는 3배 피해를 입습니다.


[ 조작법 (모바일 최적화) ]

- 스마트폰을 가로로 쥐고 조작하세요.
- 좌측 조이스틱: 부드러운 전후좌우 이동 (벽에 부딪히면 미끄러집니다)
- 우측 화면 드래그: 화면 중앙의 십자선(+)이 완벽한 총알 타격점입니다. 드래그하여 조준하세요.
- 우측 버튼: 사격(꾹 누르면 연사), 스킬, 궁극기
